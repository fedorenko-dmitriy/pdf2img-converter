== Description ==
This plugin provides conversion of PDF file when it uploads to image file.
Plugin features:
* Chooses different file image types (jpg, png, bmp, gif)
* Sets size on converted images
* Sets quantity of converted pages PDF file (from 1 to 5 first pages)
* Removes PDF file after conversion

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/pdf2img-converter` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Plugin PDF2IMG converter to configure the plugin
